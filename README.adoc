== Overview

*My-Newsroom* demonstrates how formal uncertainty quantification (Dempster-Shafer theory) and multi-agent systems can improve journalistic verification, fact-checking, and source credibility assessment.

=== The My Language Family

A suite of four interoperable dialects with *graduated trust semantics*:

[cols="1,2,2,1"]
|===
| Dialect | Purpose | Key Feature | Status

| *Me*
| Epistemic types, belief states
| `belief x: Float where confidence(0.75)`
| Complete

| *Solo*
| Systems programming, memory safety
| Affine types, arena allocation
| 40% (compiler in progress)

| *Duet*
| Human-AI co-programming
| `@synth` holes, SPARK verification
| Specification only

| *Ensemble*
| Multi-agent orchestration
| Dempster-Shafer fusion, BFT consensus
| Specification only
|===

== Project Architecture

=== The Newroom Demonstration

The flagship application: a *multi-agent newsroom* for claim verification.

[source]
----
comptime orchestrate NewsVerificationRing {
    agents: [
        ReporterAgent(domain: "politics"),
        ReporterAgent(domain: "science"),
        FactCheckerAgent(sources: [Snopes, FactCheck]),
        EditorAgent(threshold: 0.85),
    ],
    topology: Ring,
    fusion: DempsterShafer,
    consensus: Threshold(0.85),
}
----

*Key Features:*

* 50-100 specialized agents (reporters, fact-checkers, editors)
* Dempster-Shafer belief fusion for evidence combination
* Byzantine fault tolerance (33% malicious agents tolerated)
* Epistemic ledger for immutable audit trails
* Offline-first (works air-gapped)

=== Belief Fusion Example

[source,julia]
----
using MyNewsroom

# Define frame Θ = {true, false}
θ = Set(["true", "false"])

# Source A (Reuters): 85% believe true
source_a = BeliefMass(Dict(Set(["true"]) => 0.85, θ => 0.15))

# Source B (Twitter): 60% believe true
source_b = BeliefMass(Dict(Set(["true"]) => 0.60, θ => 0.40))

# Fuse beliefs
result = fuse_beliefs(source_a, source_b, Dempster)
# Result: higher confidence in "true" due to agreement
----
